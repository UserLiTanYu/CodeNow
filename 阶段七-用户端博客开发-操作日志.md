## 二、后端：新增公开博客 API

### 2.1 新建 BlogController

文件：`codenow-backend/src/main/java/com/codenow/controller/BlogController.java`

| 接口 | 路径 | 说明 |
|------|------|------|
| GET | `/api/blog/articles` | 分页查询已发布文章，支持 categoryId/tagId 筛选 |
| GET | `/api/blog/articles/{id}` | 查询文章详情，浏览量 +1 |
| GET | `/api/blog/categories` | 查询所有分类 |
| GET | `/api/blog/tags` | 查询所有标签 |

### 2.2 扩展 BlogArticleService

文件：`BlogArticleService.java` + `BlogArticleServiceImpl.java`

新增两个方法：
- `pagePublishedArticles()` — 仅查询 `status=1` 的已发布文章，置顶优先排序
- `getPublishedArticleById()` — 查询已发布文章详情，同时执行 `view_count = view_count + 1`

### 2.3 Sa-Token 配置调整

文件：`SaTokenConfig.java`

在排除路径中新增 `/api/blog/**`，使博客公开接口无需登录认证。

## 三、前端：博客前台页面

### 3.1 安装依赖

```bash
npm install marked highlight.js
```

- `marked` — Markdown 渲染为 HTML
- `highlight.js` — 代码语法高亮

### 3.2 新建 BlogLayout.vue

文件：`codenow-frontend/src/layout/BlogLayout.vue`

博客前台布局，包含：
- **顶栏**：站点名称"码上记" + 分类导航 + 管理后台入口
- **主体**：左侧主内容区 + 右侧侧边栏（标签云、关于）
- **底栏**：版权信息

### 3.3 新建 BlogHome.vue

文件：`codenow-frontend/src/views/blog/BlogHome.vue`

博客首页，文章卡片列表展示：
- 置顶标记（红色标签）
- 文章标题、摘要、分类、发布时间、阅读量、标签
- 封面图展示（如有）
- 分页组件
- 骨架屏加载状态

### 3.4 新建 BlogArticle.vue

文件：`codenow-frontend/src/views/blog/BlogArticle.vue`

文章详情页：
- 文章标题、分类、发布时间、阅读量、标签
- Markdown 内容渲染（使用 marked + highlight.js）
- 完整的 Markdown 样式（标题、代码块、表格、引用、列表等）
- 路由参数变化时自动重新加载

### 3.5 新建 BlogCategory.vue

文件：`codenow-frontend/src/views/blog/BlogCategory.vue`

按分类筛选的文章列表：
- 页面标题显示当前分类名称
- 复用文章卡片列表样式
- 分页功能

### 3.6 新建 BlogTag.vue

文件：`codenow-frontend/src/views/blog/BlogTag.vue`

按标签筛选的文章列表：
- 页面标题显示当前标签名称
- 复用文章卡片列表样式
- 分页功能

### 3.7 更新路由配置

文件：`codenow-frontend/src/router/index.js`

新增 `/blog` 路由组：
- `/blog` → 博客首页
- `/blog/article/:id` → 文章详情
- `/blog/category/:id` → 分类文章
- `/blog/tag/:id` → 标签文章

路由守卫调整：`/blog/**` 路由不检查 token，公开访问。
