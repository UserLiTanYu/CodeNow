# 扩展功能五：文件上传（阿里云 OSS）— 操作日志

> 本功能实现文章编辑时的图片上传，支持上传到阿里云 OSS 并返回可访问的 URL，替代手填封面图 URL 的方式。

---

## 一、OSS 配置集成

文件：`application.yaml`

新增配置项：
```yaml
alioss:
  endpoint: oss-cn-beijing.aliyuncs.com
  region: cn-beijing
  bucket-name: litanyu-java
  access-key-id: ${OSS_ACCESS_KEY_ID:}
  access-key-secret: ${OSS_ACCESS_KEY_SECRET:}

upload:
  allowed-types: image/jpeg,image/png,image/gif,image/webp
  max-size: 5242880
```

敏感信息通过环境变量注入，不硬编码在配置文件中。

---

## 二、添加阿里云 OSS SDK 依赖

文件：`pom.xml`

新增 `com.aliyun.oss:aliyun-sdk-oss:3.17.4` 依赖。

---

## 三、创建配置类

文件：`config/StorageProperties.java`

- `@ConfigurationProperties(prefix = "alioss")`
- 属性：endpoint、region、bucketName、accessKeyId、accessKeySecret

文件：`config/UploadProperties.java`

- `@ConfigurationProperties(prefix = "upload")`
- 属性：allowedTypes（允许的 MIME 类型）、maxSize（最大文件大小）

---

## 四、创建 StorageService 接口和 OSS 实现

文件：`service/StorageService.java`

接口方法：
- `upload(MultipartFile file)` — 上传文件，返回 URL
- `delete(String url)` — 删除文件

文件：`service/impl/OssStorageServiceImpl.java`

核心逻辑：
1. `@PostConstruct` 初始化 OSS 客户端
2. 上传时按日期组织目录：`codenow/yyyy/MM/dd/UUID.ext`
3. 调用 `ossClient.putObject()` 上传
4. 返回拼接的公开访问 URL
5. `@PreDestroy` 关闭 OSS 客户端

---

## 五、创建 UploadController

文件：`controller/UploadController.java`

| 接口 | 路径 | 说明 |
|------|------|------|
| POST | `/api/upload/image` | 上传图片，返回 `{url: "..."}` |

校验逻辑：
1. 文件是否为空
2. 文件类型是否在白名单中（JPG/PNG/GIF/WebP）
3. 文件大小是否超过 5MB
4. 调用 StorageService 上传

限流：`@RateLimit(maxCount = 10, timeWindow = 60)` 防止频繁上传

---

## 六、创建前端 ImageUpload 组件

文件：`components/ImageUpload.vue`

功能：
- 点击或拖拽上传区域
- 调用 `/api/upload/image` 上传文件
- 上传成功后显示图片预览
- 支持更换和删除图片
- 通过 `v-model` 双向绑定图片 URL

---

## 七、集成到文章编辑页

文件：`views/article/ArticleEdit.vue`

改动：
1. 封面图字段从手填 URL 改为 ImageUpload 组件
2. 编辑器工具栏新增"插入图片"按钮
3. 点击后弹出上传弹窗，上传成功后自动插入 Markdown 图片语法
