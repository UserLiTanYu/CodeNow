# 扩展功能三：评论系统 — 操作日志

> 本功能实现博客前台评论，支持楼中楼回复（多级嵌套），采用自关联表设计 + 内存树形组装算法。

---

## 一、数据库设计

新增 `blog_comment` 表，使用自关联 `parent_id` 实现树形结构：

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键自增 |
| article_id | BIGINT | 文章 ID |
| parent_id | BIGINT | 父评论 ID（顶级评论为 0） |
| content | TEXT | 评论内容 |
| nickname | VARCHAR(50) | 昵称 |
| email | VARCHAR(100) | 邮箱（不公开） |
| ip | VARCHAR(50) | 评论者 IP |
| status | TINYINT | 状态（0=待审核, 1=已通过） |
| create_time | DATETIME | 创建时间 |

建表 SQL 已追加到 `init.sql`，同时在 MySQL 中执行。

---

## 二、后端实现

### 2.1 实体类

文件：`entity/BlogComment.java`

- 字段与数据库一一对应
- `@TableField(exist = false) private List<BlogComment> children` — 非数据库字段，用于树形结构

### 2.2 Mapper

文件：`mapper/BlogCommentMapper.java`

继承 `BaseMapper<BlogComment>`，无自定义方法。

### 2.3 DTO

文件：`dto/CommentDTO.java`

- `articleId`（必填）
- `parentId`（可选，默认 0 为顶级评论）
- `content`（必填）
- `nickname`（必填）
- `email`（可选）

### 2.4 Service

文件：`service/CommentService.java` + `service/impl/CommentServiceImpl.java`

核心方法：

**`getCommentTree(articleId)`** — 树形组装算法：
1. 查出该文章所有已通过审核的评论
2. 用 `Map<Long, List<BlogComment>>` 按 parentId 分组
3. 从 parentId=0 的顶级评论开始，递归填充 children
4. 返回树形结构

**`pageComments(pageNum, pageSize, articleId)`** — 管理后台分页查询，支持按文章筛选

**`deleteWithChildren(id)`** — 递归删除评论及其所有子评论

### 2.5 Controller

文件：`controller/CommentController.java`

| 接口 | 路径 | 权限 | 说明 |
|------|------|------|------|
| GET | `/api/comments/article/{articleId}` | 公开 | 获取文章评论树 |
| POST | `/api/comments` | 公开 | 发表评论 |
| GET | `/api/comments` | 需登录 | 管理后台分页查询 |
| DELETE | `/api/comments/{id}` | 需登录 | 删除评论及子评论 |

管理端接口（GET /api/comments、DELETE）通过 `StpUtil.checkLogin()` 手动校验登录。

### 2.6 SaTokenConfig

文件：`config/SaTokenConfig.java`

排除路径新增 `/api/comments/**`，评论接口公开访问。

## 三、前端实现

### 3.1 评论表单组件

文件：`components/CommentForm.vue`

- 昵称（必填）、邮箱（选填）、评论内容（必填）
- 支持传入 `parentId`，回复时自动带上
- 提交后清空内容并触发 `success` 事件

### 3.2 评论树组件

文件：`components/CommentTree.vue`

- 递归渲染组件，显示评论内容、昵称、时间、"回复"按钮
- 子评论缩进展示（左侧边框标识层级）
- 点击"回复"后在当前评论下方展开回复表单

### 3.3 文章详情页集成

文件：`views/blog/BlogArticle.vue`

- 文章内容底部新增"评论区"
- 显示评论总数
- 顶部为发表评论表单
- 下方为评论树列表
- 发表/回复成功后自动刷新评论列表

### 3.4 管理后台评论页面

文件：`views/comment/CommentList.vue`

- 表格展示：ID、昵称、评论内容、文章 ID、父评论 ID、IP、时间
- 支持分页
- 删除按钮（删除评论及所有子评论）
- 已添加到侧边栏菜单和路由
