# 阶段四：前端核心开发 — 页面与路由 — 操作日志

> 记录本阶段从零搭建前端管理后台的全部操作步骤，供复盘复习使用。

---

## 步骤 1：配置 Vite API 代理

修改 `vite.config.js`，添加 `server.proxy` 配置：

- 将 `/api` 请求代理到 `http://localhost:8080`
- 解决前端开发时的跨域问题，无需后端额外配置 CORS

---

## 步骤 2：封装 Axios 请求工具

新建 `src/utils/request.js`，封装统一的 HTTP 请求实例：

- **基础路径**：`baseURL: '/api'`，配合 Vite 代理使用
- **请求拦截器**：自动从 localStorage 读取 Token 并放入请求头 `Authorization`
- **响应拦截器**：统一处理返回格式，`code !== 200` 时弹出 `ElMessage.error` 错误提示
- **Token 失效处理**：检测到 token 相关错误时，清除本地 Token 并跳转登录页

---

## 步骤 3：创建 Pinia 用户状态管理

新建 `src/stores/user.js`，定义用户 Store：

- `token`：响应式变量，从 localStorage 初始化
- `userInfo`：当前用户信息
- `login(username, password)`：调用登录接口，存储 Token 到 localStorage
- `logout()`：调用登出接口，清除 Token 和用户信息
- `fetchUserInfo()`：调用 `/api/auth/me` 获取当前用户信息

---

## 步骤 4：注册 Element Plus 全局组件

修改 `src/main.js`：

- 全局注册 Element Plus 组件库
- 引入中文语言包 `zhCn`，使组件默认显示中文
- 引入 Element Plus 样式文件

---

## 步骤 5：配置 Vue Router 路由

重写 `src/router/index.js`，定义完整路由表：

| 路径 | 页面 | 说明 |
|------|------|------|
| `/login` | LoginView | 登录页 |
| `/` | MainLayout | 主布局（重定向到 `/articles`） |
| `/articles` | ArticleList | 文章列表 |
| `/articles/edit/:id?` | ArticleEdit | 文章编辑（id 可选，无 id 为新建） |
| `/categories` | CategoryList | 分类管理 |
| `/tags` | TagList | 标签管理 |

添加路由守卫 `beforeEach`：检查 localStorage 中的 Token，未登录自动重定向到 `/login`。

---

## 步骤 6：实现主布局

新建 `src/layout/MainLayout.vue`，使用 Element Plus 的 Container 组件：

- **左侧菜单栏（el-aside）**：深色背景，显示"码上记"Logo 和三个菜单项（文章管理、分类管理、标签管理）
- **顶部导航栏（el-header）**：显示当前页面标题，右侧下拉菜单显示用户名和退出登录
- **右侧内容区（el-main）**：灰色背景，渲染 `<router-view />` 子路由

菜单使用 `el-menu` 的 `router` 模式，点击菜单项自动跳转对应路由。

---

## 步骤 7：简化 App.vue

将 `App.vue` 从脚手架默认模板简化为只包含 `<router-view />`，布局逻辑全部交给 MainLayout。

---

## 步骤 8：实现登录页

新建 `src/views/login/LoginView.vue`：

- 渐变紫色背景居中显示登录卡片
- 表单包含用户名和密码输入框，支持回车提交
- 使用 `el-form` 的 `rules` 进行表单校验（必填项）
- 登录按钮带 loading 状态，防止重复提交
- 登录成功后跳转到首页，显示"登录成功"提示

---

## 步骤 9：实现文章列表页

新建 `src/views/article/ArticleList.vue`：

- **工具栏**：左侧"新增文章"按钮，右侧分类/标签下拉筛选框
- **表格**：展示标题（含置顶标记）、分类、标签（el-tag 标签组）、状态、创建时间、操作列
- **操作列**：编辑、发布/下架、删除按钮
- **分页**：底部 el-pagination 分页组件，支持翻页
- **加载状态**：表格使用 `v-loading` 指令
- **删除确认**：使用 `ElMessageBox.confirm` 二次确认

---

## 步骤 10：实现文章编辑页

新建 `src/views/article/ArticleEdit.vue`：

- 使用 `el-form` 表单，包含标题、分类下拉、标签多选、摘要、封面图 URL、Markdown 内容
- Markdown 编辑器使用 `md-editor-v3` 组件（命名导入 `{ MdEditor }`）
- 支持新建和编辑两种模式，通过路由参数 `id` 区分
- 编辑模式下自动加载文章数据填充表单
- 底部三个按钮：取消（返回上一页）、保存草稿、发布
- 表单校验：标题、内容、分类为必填项

---

## 步骤 11：实现分类管理页

新建 `src/views/category/CategoryList.vue`：

- 表格展示分类名称、描述、排序号、创建时间
- "新增分类"按钮打开 el-dialog 弹窗
- 弹窗内表单包含名称（必填）、描述、排序号
- 编辑和新增共用同一个弹窗，通过 `isEdit` 标志区分
- 删除操作使用 `ElMessageBox.confirm` 确认

---

## 步骤 12：实现标签管理页

新建 `src/views/tag/TagList.vue`：

- 顶部输入框 + "新增标签"按钮，支持回车添加
- 表格展示标签 ID、名称、创建时间
- 操作列只有删除按钮
- 删除使用 `ElMessageBox.confirm` 确认
